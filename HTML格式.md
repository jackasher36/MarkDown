## 超链接

```html
//百度搜索
<form action="https://www.baidu.com/s">
    <input type="text" name="wd">
    <button>Search</button>
</form>

//京东搜索
<form action="https://search.jd.com/search">
    <input type="text" name="keyword">
    <button>Search</button>
</form>
```

## FiledSet

```html
<fieldset>
    <legend>主要内容</legend>
    <form action="https://www.baidu.com/s">
        <input type="text" name="wd">
        <button>Search</button>
    </form>

    <form action="https://search.jd.com/search">
        <input type="text" name="keyword">
        <button>Search</button>
    </form>
</fieldset>
```
